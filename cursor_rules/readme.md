## Development Commands

```bash
# Setup
bundle install && yarn install
rails db:create db:migrate db:seed
./bin/dev                        # Start Rails + Vite

# Database
rails db:migrate                 # Run migrations
rails db:seed                    # Seed data
rails db:reset                   # Full reset

# Testing
rails test                       # Backend tests
yarn test                        # Frontend tests
yarn cypress:open                # E2E tests

# Generators
bin/rails g model ModelName field:type
rails g migration AddFieldToModel field:type
```
